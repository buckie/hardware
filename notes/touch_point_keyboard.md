# Notes on german trackpoint keyboard circuit

## Trackpoint Pin Layout

### Continuity Measurements

    (underside)
    ---------------------------
    |    (1) (2) (3)          | 
    |      (4) (5) (6)        | 

Vcc (5v) to:
* 3 == 0
* 1, 2, 5 & 6 == .250
* 4 == .325

GND to:
* ALL == No Connection
